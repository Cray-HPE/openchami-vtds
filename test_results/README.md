# Test Results Directory

This directory is home to test result reported by integration test runs. Each directory found here contains the full set of results generate by deploying and running tests on OpenCHAMI using vTDS. Each directory is named using the version string corresponding to the version of OpenCHAMI Release under test, combined with a date / time-stamp to differentiate different runs of the same version.
